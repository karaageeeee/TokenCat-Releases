# Cat animation frames

PNG frames for the menu bar cat animation live here. Replace files with the same
names to customize the animation.

Expected file names:

- `running_0.png` ... `running_3.png`
- `tired_0.png` ... `tired_3.png`
- `sleepy_0.png` ... `sleepy_3.png`
- `exhausted_0.png` ... `exhausted_3.png`
- `collapsed_0.png` ... `collapsed_3.png`
- `unavailable_0.png` ... `unavailable_3.png`

The current frames are generated original pixel-art assets, not copied RunCat
artwork. Each animation frame is a 66x54 RGBA PNG. Transparent pixels keep
`#00ff00` as their underlying RGB value so the frames are easy to inspect or
chroma-key if needed.
