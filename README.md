# TokenCat v0.2.14

Notarized macOS release of TokenCat.

## Download

Download `TokenCat-2.14.zip`, unzip it, and move `TokenCat vol.2.14.app` to your Applications folder.

## Requirements

- macOS 14 or later
- Claude Code login, if using Claude Code usage
- Codex login, if using Codex usage
